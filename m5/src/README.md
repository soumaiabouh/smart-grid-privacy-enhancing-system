# Smarter Metering System

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the required dependencies.

```bash
pip install pycryptodome
pip install phe
pip install tqdm

pip install openpyxl

npm install mongoose
pip install pymongo
npm install express
```

## Usage

1. To run the MDMS UI, the user has to navigate to the m5/src/mdms-ui folder and run:

```bash
 node app.js
```
The username and password are: 
```
username: adminTester 
password: password
```

2. For the interactions between the PES and other components, run the `main.py` file.

3. For the UCS, run the `user-centric-system.py` file.
